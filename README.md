# Lead Generation Parser v2 (2GIS)

Терминальный скрапер нового поколения для сбора бизнес-баз (лидогенерации) из 2GIS (преимущественно Кыргызстан: Бишкек, Ош и другие города).
Это решение построено на модульной архитектуре с использованием Selenium для сбора закрытых данных (телефоны, соц. сети) и сохранения результатов напрямую в локальную базу данных (SQLite).

## Особенности

- **Единая база данных:** Все собранные лиды сохраняются в SQLite (`leads.db`). Нет больше потерянных CSV или кучи мелких файлов.
- **Умная дедупликация:** При повторных запусках новые лиды добавляются, а старые не дублируются (через внутренний ID 2GIS).
- **Сбор ценных контактов:** Бот "кликает" в браузере, чтобы достать номера телефонов, Instagram, сайты и электронные почты.
- **Интерактивное меню CLI:** Удобный выбор городов и категорий прямо в терминале.
- **Интегрированный Фильтр & Экспорт:** Выгружай в готовые форматы Excel (`.xlsx`) или CSV. Можно отфильтровать (например, "выгрузить только те, у кого есть телефон").

## Структура проекта (Модульная)

Вся бизнес-логика и сбор данных отделены от интерфейса. Это сделано специально для того, чтобы в будущем этот же код можно было легко обернуть в Web-интерфейс (через Flask или FastAPI).

```
lead_gen/
├── cli/                 # Интерфейс командной строки
│   └── main.py          # Интерактивное меню (запускать отсюда)
├── core/                # Главная логика
│   ├── models.py        # Структура данных Лида (dataclass)
│   └── scraper.py       # Движок Selenium (парсинг 2GIS)
├── db/                  # Работа с хранилищем и экспортом
│   ├── database.py      # Сохранение и выборка из SQLite через pandas
│   └── exporter.py      # Конвертация в Excel и CSV
├── exports/             # (Создастся автоматически) Сюда падают готовые выгрузки
├── leads.db             # (Создастся автоматически) Локальная база с вашими лидами
├── config.py            # Общие настройки (список городов, категорий)
└── requirements.txt     # Зависимости Python
```

## Установка и запуск

1. Перейдите в папку проекта:
```bash
cd lead_gen
```

2. Установите зависимости (потребуется Python и установленный Google Chrome):
```bash
pip install -r requirements.txt
```

3. Запустите консольное меню:
```bash
python cli/main.py
```

## Как использовать

После запуска `python cli/main.py` появится меню:

### 1. Скрапинг данных
Выберите пункт `1` (Новый поиск). Программа предложит выбрать город, затем категорию бизнеса (например, Стоматологии или Рестораны) и количество страниц (в 2GIS по умолчанию много результатов разбиты по страницам). 
Скрипт откроет невидимый браузер Chrome, соберет данные и запишет их в `leads.db`. 

**Защита от сбоев:** Скрипт сохраняет лиды "пачками" (batches). Если вы решите остановить поиск досрочно сочетанием `Ctrl+C` (Или пропадет интернет), уже собранные лиды не потеряются.

### 2. Экспорт (Фильтрация и Выгрузка)
Выберите пункт `3` (Фильтр и экспорт). Вы сможете посмотреть какие данные собраны и выгрузить их в:
*   Полный Excel / CSV файл.
*   Excel только с теми компаниями, для которых **смогли найти телефон**.
*   Excel отфильтрованный по конкретной категории или определенному городу.
Готовые файлы сохраняются в папку `exports/`.

> **Внимание к Капчам:** При интенсивном скрапинге 2GIS может показать анти-бот капчу. Скрипт попытается подождать, но старайтесь не ставить запросы на тысячи страниц разом (или доработайте логику задержек в `config.py` по необходимости).
